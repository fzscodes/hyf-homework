web development as an industry

Examples of dramatic transformation in communication
and commerce led by web development include e-com-
merce.Online aution sites such as eBay have changed
the way consumes find and purchase goods and servic-
es.Online retails such as Amazon.com and Buy.com
(among many others)have transformed the shopping
and bargain-hunting experience for many customers.

practical web development

Since the commercialisation of the web,web development has been growing
industry.The growth of this industry is being driven by businesses wishing to use
their website to sell products and services to the customer.[1]

 

There is open source software for web development like BerkelelyGB,Glass Fish,
LAMP(Linux,Apache,MYSQL,PHP)stack and Perl/Plack.This has kept the cost of
learning web development to a minimum.Another contributing factor to the
growth of the industry has been the rise of easy-to-use WYSIWYS web devlop-
ment software,such as Adobe Dreamweaver,BlueGriffon and Microsoft Visual
Studio.Knowledge of HyperText Markup Language(HTML) or of programming
languages is still required to use such software,but the basics can be learned and
implemented quickly with the help of help files,technical books,internet tutorials,
or face to face tutorial.

 

An ever-growing set of tool and technologies have helped developers build more
dynamic and interactive websites.Futher,web developers now help to deliver
applications as web services which were traditionally only available as applica-
tionson a desk-based computer.Examples can be seen with the rise
of cloud services such as Adobe Creative Cloud,Dropbox and Google Docs.These
web services allow users to interact with applications from many locations,
instead of being tied to specific workstation for their application environment.
